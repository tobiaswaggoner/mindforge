# Types Package

Geteilte Typdefinitionen für Konsistenz zwischen Python und TypeScript.

## Struktur

```
types/
├── python/          # Python dataclasses/Pydantic models
└── typescript/      # TypeScript interfaces
```

## Kerntypen

- `QuestionCluster` - Kanonische Frage-Definition
- `QuestionVariant` - Generierte Variante
- `Subject` - Fach/Themenbereich
- `GenerationSession` - Tracking-Session für AI-Generierungen
