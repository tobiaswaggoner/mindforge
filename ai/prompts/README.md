# LLM Prompts

Dieses Verzeichnis enthält die Prompts für die AI-gestützte Content-Generierung.

## Geplante Prompts

- `variant-generation.md` - Prompt für Fragen-Varianten-Generierung
- `quality-check.md` - Prompt für Qualitätsprüfung
- `cluster-creation.md` - Prompt für neue Question-Cluster

## Verwendung

Prompts werden vom Python-Backend geladen und an den konfigurierten LLM-Provider gesendet.
