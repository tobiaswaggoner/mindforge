# Create pull request

Bitte erstelle einen Pull Request für alle Änderungen auf diesem Branch

## Anweisungen

1. Stelle sicher, dass wir uns nicht auf dem "main" Branch befinden. Wenn doch: Brich die Verarbeitung ab.
2. Stelle sicher, dass alle Änderungen comitted sind. Wenn es nicht comittede Änderungen gibt, breche die Verarbeitung mit einem Hinweis ab.
3. Stelle sicher, dass alle Commits des Branches gepusht sind. Pushe diese wenn das nicht der Fall ist.
4. Schau Dir den Diff des aktuellen Branches im Vergleich zu main an.
5. Erzeuge den Pull Request, der diese Änderungen beschreibt.
