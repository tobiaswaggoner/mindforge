# Create git commit

Bitte erstelle einen git commit

## Anweisungen

1. Stelle sicher, dass wir uns nicht auf dem "main" Branch befinden. Wenn doch: Brich die Verarbeitung ab.
2. Stelle sicher, dass wirklich alle Änderungen "gestaged" sind. Wenn nicht, hole das nach.
3. Schau Dir alle gestagten Änderungen an.
4. Erstelle eine gute git commit message (bitte keine Emojis)
5. Comitte die Änderungen.
