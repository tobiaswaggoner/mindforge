# Merge main

Der Main branch hat eventuell neue Änderungen. Wir wollen diese Forward Mergen auf unseren aktuellen Branch.

## Anweisungen

1. Stelle sicher, dass wir uns nicht auf dem "main" Branch befinden. Wenn doch: Brich die Verarbeitung ab.
2. Stelle sicher, dass es keine nicht comitteten Änderungen auf unserem Branch gibt. Wenn doch: Brich die Verarbeitung ab.
3. Hole alle Änderungen vom Remote des Main Branch 
4. Merge den Main Branch in den aktuellen Branch
